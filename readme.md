# 尚融宝

## 项目介绍

尚融宝是一个**网络借贷信息中介服务平台**，致力为高成长人群提供专业的线上信贷及出借撮合服务。



## 项目文件说明

```
├── readme.md       （说明文档）
├── hfb             （第三方资金管理机构源文件）
├── pic             （说明文档图片）
├── srb             （后端业务代码）
├── srb-admin       （前端后台页面）
└── srb-site        （前端业务页面）
```
